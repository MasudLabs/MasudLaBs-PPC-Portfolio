# MasudLaBs PPC Portfolio

**Professional Amazon PPC Tools, Scripts & Templates**

A curated collection of practical tools and resources built for Amazon Advertising Specialists and Virtual Assistants. Focused on automation, data analysis, negative keyword management, bid optimization, and campaign structuring.

---

## 🚀 About This Portfolio

This repository showcases hands-on skills in:

- Amazon PPC campaign architecture & structure
- Keyword research & negative keyword strategy
- Search term report analysis & insights
- Performance metrics mastery (ACoS, ROAS, CTR, CVR, TACoS)
- Automation & scripting for efficiency
- Clear documentation and process building

**Objective**: Build real, usable tools that help Amazon sellers and agencies save time, reduce wasted ad spend, and make smarter bidding & keyword decisions.

---

## 📁 Projects & Tools

### 1. Keyword Research Tool / Negative Keyword Generator
**Status**: Planned

Automated tool that:
- Generates Broad, Phrase, and Exact match keywords from seed terms
- Suggests high-intent long-tail variations
- Analyzes search term data to recommend negative keywords (auto-negative + manual review)
- Outputs clean CSV ready for Sponsored Products campaigns

### 2. Search Term Report Analyzer
**Status**: Planned

Powerful analyzer that processes Amazon Search Term Reports and delivers:
- Top converting search terms
- High spend / low conversion terms (wasted budget)
- Strong negative keyword candidates
- Performance trends over time
- Actionable recommendations with priority scoring

### 3. ACoS / ROAS Calculator + Optimizer
**Status**: Planned

Advanced calculator featuring:
- Real-time ACoS & ROAS calculation
- Break-even ACoS analysis
- Target ACoS / ROAS optimizer
- Bid adjustment recommendations based on performance
- Scenario modeling ("What if I increase bids by X%?")

### 4. Bid Strategy Simulator
**Status**: Planned

Simulates different bidding strategies and forecasts impact:
- Dynamic Bids (up & down)
- Fixed Bids
- Rule-based / Portfolio bidding
- Projects changes in ACoS, ROAS, impressions, clicks, and sales

### 5. Campaign Structure Templates (with Documentation)
**Status**: Planned

Ready-to-deploy campaign structures for different objectives:
- Brand Defense / Branded Terms
- Category Targeting & Broad Match
- Competitor Conquesting
- Long-tail Harvesting
- Seasonal / Event-based campaigns

Each template includes full structure explanation + best practices.

### 6. Automated Reporting Script
**Status**: Planned

Python script that:
- Pulls key metrics from exported reports
- Generates clean daily/weekly performance summaries
- Highlights trends, anomalies, and opportunities
- Can be scheduled or run manually
